# Game Rock-Paper-Scissors

Dự án: Trò chơi "Kéo - Búa - Bao" (Rock-Paper-Scissors).

Mục tiêu
- Triển khai trò chơi cơ bản để người chơi đối đầu với máy hoặc hai người chơi.

Cấu trúc (ví dụ)
- `main.py` hoặc `index.js` — entrypoint của trò chơi
- `src/` — mã nguồn
- `assets/` — hình/âm thanh (nếu có)

Chạy nhanh (gợi ý)
- Nếu là Python:
  - Tạo môi trường: `python -m venv .venv`
  - Kích hoạt (PowerShell): `.\.venv\Scripts\Activate.ps1`
  - Cài dependencies: `pip install -r requirements.txt` (nếu có)
  - Chạy: `python main.py`

- Nếu là Node.js:
  - Cài: `npm install`
  - Chạy: `node index.js` hoặc `npm start`
